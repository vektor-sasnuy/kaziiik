## 🚀 ШВИДКИЙ СТАРТ

```bash
# 1. Мігрувати БД
python manage.py makemigrations cases
python manage.py migrate

# 2. Додати тестові дані
python populate_cases.py

# 3. Запустити сервер
python manage.py runserver

# 4. Перейти на http://localhost:8000/cases/
```
